# ai-powered-app-workshop

This workshop provides an introduction to building AI powered Shiny apps. AI powered apps use LLMs, perhaps augmented with tools, MCP servers, and more, to provide sophisticated functionality.

This workshop will show you how to build AI agents packaged as Shiny apps. Shiny apps are easy to use, share, 

After taking this workshop, learners will be able to:

1. Use the ellmer package to create a chat session in R powered by a common LLM.
1. Safely store AI authorization credentials in a .Renviron file.
1. Change an LLM's behavior within a chat session by writing a system prompt.
1. Explain how LLMs and tools interact to create an AI agent.
1. Use the ellmer package to provide a tool to a chat session to create a simple agent.
1. Use shinychat to add a simple AI agent to a Shiny app.
1. Deploy an AI powered shiny app to Posit Connect.
1. Add secret variables to Connect, to arrange for LLM access for your published Shiny app.
